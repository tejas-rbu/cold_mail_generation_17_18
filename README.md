## 🧠 Project Overview

The Cold Email Campaign Writer is an AI-powered multi-agent system designed to automatically generate professional cold emails for business outreach. The system takes inputs such as industry, product, and optional target role, and generates a complete workflow including buyer persona creation, personalized email generation, and AI-based evaluation of the email quality.

## ⚙️ Features

This system includes a multi-agent architecture consisting of a persona generator, email writer, and evaluation judge. It supports automated cold email generation, AI-based scoring and feedback, a Streamlit-based interactive user interface, and history tracking of previously generated emails. The UI is designed with a modern SaaS-style layout for better user experience.

## 🏗️ System Architecture

The system is built using three main agents. The Persona Agent generates a detailed buyer persona based on input. The Email Agent creates a personalized cold email using the persona. The Judge Agent evaluates the email and provides a structured score along with feedback.

## 🧩 Workflow

The workflow begins with user input in the form of industry and product. This is followed by persona generation, then email generation, and finally evaluation of the email. The results are displayed in a Streamlit UI in a structured format.

## 🛠️ Tech Stack

The project is built using Python, Streamlit, Groq API for LLM integration, dotenv for environment management, and JSON processing for structured data handling.

## 🚀 How to Run

To run the project, first install dependencies using pip install -r requirements.txt. Then start the application using streamlit run app.py.

## 🔑 Environment Setup

A .env file must be created containing the Groq API key in the format GROQ_API_KEY=your_api_key_here.

## 🌐 Deployment

The project can be deployed using platforms such as Railway, Render, or Streamlit Cloud.

## 📊 Output Includes

The system generates a buyer persona, a personalized cold email, an AI evaluation score, and feedback for improvement.
